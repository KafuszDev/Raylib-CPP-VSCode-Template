# Raylib C++ VSCode Template

A simple Raylib C++ template for Visual Studio Code for game development.

Ready-to-use setup for building Raylib games in C++ without manual configuration.

---

## Features

- Raylib C++ project structure
- Visual Studio Code configuration (tasks, debug, IntelliSense)
- GCC (MinGW / w64devkit) build system
- Simple one-click build and run setup
- Minimal and easy to modify

---

## Requirements

- Windows
- Visual Studio Code
- C++ compiler (GCC / MinGW / w64devkit)
- Raylib

---

## Important Setup

This template assumes a fixed directory structure:

- Raylib: `C:/raylib/raylib`
- Compiler (w64devkit): `C:/raylib/w64devkit`

If your setup is different, update paths in:
- `tasks.json`
- `launch.json`
- `c_cpp_properties.json`

---

## Getting Started

1. Clone or download this repository
2. Open folder in Visual Studio Code
3. Build the project (Ctrl + Shift + B)
4. Run and debug in VS Code
5. Start developing your Raylib game

---

## Notes

This is a minimal template focused on simplicity and fast setup, not portability.
